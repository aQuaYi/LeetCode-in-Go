# [168. Excel Sheet Column Title](https://leetcode.com/problems/excel-sheet-column-title/)

## 题目

        Given a positive integer, return its corresponding column title as appear in an Excel sheet.

For example:
    1 -> A
    2 -> B
    3 -> C
    ...
    26 -> Z
    27 -> AA
    28 -> AB
